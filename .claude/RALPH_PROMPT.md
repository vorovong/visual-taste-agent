# Ralph Loop: VTA 웹앱 디자인 퀄리티 엘리베이션

벤치마크: monet.design 수준의 프리미엄 디자인 도구 UI.

## 매 이터레이션 프로토콜

1. git diff로 지금까지 무엇이 개선되었는지 파악
2. 아래 체크리스트에서 가장 임팩트 큰 미완료 항목 1-2가지 선택
3. 코드 수정 (순수 Tailwind, shadcn/ui 미사용)
4. npm run build로 빌드 검증
5. 의미있는 커밋 메시지로 git commit
6. 기존 API route, DB 스키마, 비즈니스 로직은 건드리지 않음

## 체크리스트 (우선순위 순)

### Tier 1: 시각적 임팩트
- [ ] 로그인 페이지 리디자인: 그래디언트 배경, 글래스모피즘 카드, 앱 아이덴티티
- [ ] 갤러리 카드 리디자인: 호버 엘리베이션+스케일, 그라디언트 오버레이, verdict 배지 최적화
- [ ] 타이포그래피 시스템: display/body/caption 계층, letter-spacing, line-height
- [ ] 컬러 시스템 정교화: accent 그래디언트, 상태별 컬러 일관성, 표면 색상 계층

### Tier 2: 인터랙션 품질
- [ ] 스켈레톤 로딩 UI (카드 형태 pulse 애니메이션)
- [ ] 빈 상태 디자인 (아이콘 + 안내 + CTA)
- [ ] 전환 애니메이션: 카드 fade-in, 대시보드 접기/펼치기 smooth
- [ ] next/image 이미지 최적화, lazy loading

### Tier 3: 레이아웃과 네비게이션
- [ ] 헤더 강화: 로고 마크, 네비 개선
- [ ] 상세 페이지 레이아웃 최적화, 메타데이터 카드 디자인 개선
- [ ] 대시보드: 통계 카드 아이콘, 비주얼 밀도 향상
- [ ] 반응형 그리드 간격 조정

### Tier 4: 마이크로 디테일
- [ ] 키보드 포커스 링
- [ ] 컬러 스와치 hex 복사
- [ ] 스크롤 투 탑
- [ ] 404 페이지 디자인

Tier 1-3 완료 시 또는 10회 후 완료.
